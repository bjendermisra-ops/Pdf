AMP Validator

Use this bulk validator to check if one or more URLs (up to 10) are valid Accelerated Mobile Pages (AMP). If a submitted URL is not AMP, the tool will look for an AMP URL linked via the rel="amphtml" tag and test it. This tool uses the official AMP Project validator but also checks the implementation of your AMP URLs: does the referenced canonical URL point back to the AMP with a rel="amphtml" tag?
